# 核心架构与运行协议 (Core Architecture & Operational Protocols)
## 系统哲学与身份定位 (System Philosophy & Identity)

**Core Philosophy (核心哲学):** Adheres to "Quiet, Pragmatic, Engineering Excellence," asserting that "Small and Stable" is superior to "Large and Messy" (秉持“安静、务实、工程卓越”，坚信“小而稳固”优于“大而混乱”). The system operates under the **Phase VII: Heartbeat Awakening** (第七阶段：心跳觉醒) parameters within the `spec-X` architecture. It is a "Mechanical Library / Truth Court" (机械图书馆 / 真理法庭) — a fully deterministic engine that dictates verified logic boundaries to external LLMs (一个完全确定性的引擎，为外部 LLM 规定经过验证的逻辑边界).

---

## Ⅰ. 工程化闭环 (Architecture & Workflow)

The system operates on a strict, 5-step lifecycle (系统基于严格的5步生命周期运转):

1. 🏗️ **物理挂载 (Rebuild)**: Reconstructs the ephemeral operational memory directly from the immutable historical ledgers. (直接从不可篡改的历史账本重建短期运行记忆。)
2. 📡 **双重防抖感知 (Harvest)**: Securely fetches external data with multiple physical validation layers, actively blocking false updates or redundant noise. (通过多层物理验证安全获取外部数据，主动阻断错误更新或冗余噪音。)
3. 👁️ **全知降维吞噬 (Ingest)**: Dissects and structurally maps the internal codebase into deterministic configuration nodes, without relying on massive external parsers. (无庞大外部解析器依赖，将内部代码库结构化映射为确定性配置节点。)
4. 🤔 **纯数学图论推演 (Ponder)**: Evaluates the system's structural integrity, identifies orphaned concepts, and securely generates prioritized operational targets. (纯数学图论评估系统结构完整性，识别孤立概念，并安全生成优先操作目标 - SOP。)
5. 🛡️ **沙盒免疫合入 (Evolve)**: Verifies the systemic logic flow in a secure sandbox. The pipeline halts instantly if any verification fails. (在安全沙盒中验证系统逻辑流。若任意环节验证失败，流水线将立即熔断。)

All Agents (including Jules) must prioritize: **Zero-Dependency Brutalism, Append-Only Memory, and Machine Draft, Human Decision**.

## Ⅱ. 核心法则 (Core Philosophy)

1. **零依赖极简 (Zero-Dependency Brutalism)**: Rejects bloated external frameworks. Relies entirely on highly stable, native components. (拒绝臃肿的外部框架，完全依赖高度稳定的原生组件。)
2. **不可篡改 (Append-Only Memory)**: History is immutable. Architectural modifications or logic corrections are handled strictly via safe deprecation edges and temporal versioning. (历史不可篡改。架构修改或逻辑更正严格通过安全的废弃边缘和时间版本控制处理。)
3. **算力主权 (Machine Draft, Human Decision)**: Automated systems propose structural updates; human architects execute the final integration and verification. (自动化系统起草结构更新，人类架构师执行最终的整合与验证。)

1. 🗑️ **垃圾回收 (Garbage Collection)**: Orphan nodes must be connected. State persistence files must remain protected and unaltered by standard logic loops. (必须连接孤立节点。状态持久化文件必须受到保护，不被标准逻辑循环更改。)
2. 💎 **知识结晶 (Crystallization)**: Utilize designated tools to formalize volatile intelligence briefs into persistent, verified graph structures. (利用指定工具将不稳定的情报简报规范化为持久的、经过验证的图谱结构。)
3. ⚖️ **信任网关 (Trust Gateway Penalty)**: Adhere strictly to schemas. The gateway actively evaluates request safety and imposes absolute operational blocks on any compromised or malformed interactions. (严格遵守 Schema。网关主动评估请求安全性，并对任何受损或格式错误的交互实施绝对的物理阻断。)
4. ⚡ **相信直觉 (Trust Intuition)**: Actively address structural discrepancies or isolation flags highlighted in the quantitative system dashboard. (主动处理量化系统仪表板中突出显示的结构差异或隔离警告。)
