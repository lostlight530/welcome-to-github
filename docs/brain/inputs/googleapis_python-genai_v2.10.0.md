# 🎯 监控目标 (Target)
谷歌的 Python GenAI SDK (googleapis/python-genai)

# 🚀 新版本发布 (New Release)
发布了 v2.10.0 版本，主要新增了对 Agent Platform MCP 的支持以用于异步生成内容流，为交互和模型添加了禁用安全策略的计算机使用选项，增加了步骤的使用字段，并引入了视频生成和响应格式参数

# 💡 架构洞察 (Architectural Insight)
引入了 MCP (Model Context Protocol) 支持，增强了生成式 AI 流程的互操作性并扩展了代理平台的集成能力，同时赋予了开发者在安全控制和媒体生成方面更大的配置自由度
