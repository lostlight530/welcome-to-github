# 阶段四：绝对决定论确立 (Phase IV: Absolute Determinism)

## 概览 (Overview)

The system retreated to the lowest levels of standard libraries, pursuing absolute physical stability. It completely abandoned any internal reliance on the fuzzy reasoning of Large Language Models (LLMs).

## 核心突破 (Key Breakthroughs)

1.  **The Lobotomy Protocol (脑白质切除协议)**
    Dynamic states (e.g., `.db` files, `.json` caches) were forcibly excluded from Git tracking (`.gitignore`), maintaining absolute purity in the tracked source code.
2.  **Mechanical Library (机械图书馆)**
    Established the "Machine Draft, Human Decision" operational flow. AI was downgraded to mere data transporters ("Hounds"), while the system itself operated strictly on pure mathematical and physical laws.
3.  **Bilingual Mandate (双语强制)**
    Architecture and technical implementations were mandated to be strictly bilingual (Chinese/English) to guarantee absolute global accessibility.
4.  **4D Temporal Graph & Anti-Shake Radar (4D时空图谱与防抖雷达)**
    Transitioned to a temporal SQLite graph utilizing `valid_at` and `invalid_at` for non-destructive soft-deletion. Concurrently, a "Double-Clutch Anti-Shake" mechanism was implemented in `harvester.py` (stripping noise -> SHA-256 -> `difflib`) to ensure pristine, hallucination-free external knowledge ingestion.
