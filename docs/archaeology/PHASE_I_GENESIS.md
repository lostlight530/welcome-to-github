# 第一纪元：创世与孤岛期 (Phase I: Genesis & Isolation)

> **"In the beginning, it was just a sandbox. Static, quiet, and isolated."**
> **Epoch Marker**: 2026-02-14

## 1. 历史背景 (Historical Context)
这是 NEXUS CORTEX 的起源阶段。最初的愿景仅仅是建立一个用于前端展示和基础文档托管的静态试验场。在这个阶段，系统缺乏动态感知能力，完全依赖于人工提交。

## 2. 核心架构 (Core Architecture)
- **Static Frontend**: 部署在 GitHub Pages 上的静态 HTML/JS 页面，使用原生 ES Modules 和 CDN 加载的 Tailwind CSS 保证了热重载能力。
- **Isolated Components**: 项目初期的模块相互孤立，缺乏统一的认知中枢。文件和文档散落各处，作为纯粹的静态结构存在。

## 3. 局限性与突破口 (Limitations & The Breakthrough)
由于缺乏动态数据集成，所有的知识更新都需要高昂的人工成本。这促使我们开始构思一个能够自主吸收和沉淀架构经验的闭环系统，为随后的“混沌纪元”埋下了伏笔。
